# Blitz

Fast, simple, and efficient state management for Solid.

```bash
## Getting started is easy!
# npm
npm i @dimensionhq/blitz

# yarn
yarn add @dimensionhq/blitz

# pnpm
pnpm i @dimensionhq/blitz
```
